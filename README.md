## ⚡ Quick Start

### Prerequisites
- Docker and Docker Compose
- Python 3.9+
- Git

### Installation

```bash
# Clone repository
git clone https://github.com/Worldwidebro/iza-os-project-management-synthesis-bot.git
cd iza-os-project-management-synthesis-bot

# Setup environment
./scripts/setup.sh

# Start services
docker-compose up -d

# Verify installation
./scripts/health.sh
```
